ReactiveSearch UI Components

ReactiveSearch provides 20+ pre-built components for building amazing search UIs with Elasticsearch. Try the quick start or browse components.

Bring your own components

You can use your own design components and make them reactive with ReactiveComponent.

Appbase.io

Appbase.io is a hosted realtime Elasticsearch service. Find more about it in the docs.

ReactiveMaps UI Components

ReactiveMaps adds geospatial UI components for Maps. Try the quick start or browse components.

React Native UI components

ReactiveSearch and ReactiveMaps are available in preview for React Native. Browse the Native docs here.
